## Архитектура приложения:

Приложение состоит из трех страниц, которые выводятся через элемент Frame отдельной (главной) страницы MainWindow.xaml. На этой же странице расположены кнопки для навигации с номерами заданий. При нажатии на них создается новый экземпляр страницы, соответствующей выбранному заданию.

При наведении на поля для ввода данных появляется всплывающая подсказка. Вычисления осуществляются при нажатии на кнопку "Вычислить", очистить введенные значения можно по кномке "Очистить".
